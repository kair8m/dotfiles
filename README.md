# Keyboard shortcuts

## vim

`:Files` toggle fzf dialogue

`gcc` - comment line

`gc` - uncomment line

`Ctrl + n` - toggle nerd tree

`Ctrl + f` toggle fzf dialogue

###NERD TREE guides

`m` - open dialogue

in dialogue

`a` - create file or directory

`d` - delete file or directory
