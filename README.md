# Keyboard shortcuts

## vim

`gq` - format selection

`gcc` - comment line

`gc` - uncomment line

`Ctrl + n` - toggle nerd tree

`Ctrl + p` find file in project

`Ctrl + f` find using content in project

###NERD TREE guides

`m` - open dialogue

in dialogue

`a` - create file or directory

`d` - delete file or directory
